属性

activeId { String } #

激活项的 Id，必须是 items 项的 id 属性

默认值: ''

style { Object } #

自定义样式

默认值: null

items { Object } #

tab 项

默认值: [{}]

activeOpacity { Number } #

tab 项点击时透明度

默认值: 1

onPress { Function } #

改变激活项时的回调，点击项的 id 会作为参数传入。

默认值: NOOP
